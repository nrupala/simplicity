5 Key Components

Vespa AI Retrieval

  • 200B+ URL web index
  • 10K+ updates/second
  • Hybrid dense + sparse search
  • ML-based reranking
  • Fine-grained content chunking

Model Orchestration

  • PPO-RL based routing
  • 20+ models simultaneously
  • Cost-latency-quality balance
  • Model-agnostic design
  • Vendor risk mitigation

ROSE Inference

  • Custom Python/PyTorch core
  • Rust serving layer
  • TensorRT-LLM optimization
  • Speculative decoding
  • Prefill-decode disaggregation

Perplexity Computer

  • 19-model orchestration
  • 100+ MCP tool integrations
  • Agentic workflows
  • Sequential planning
  • Parallel task execution
